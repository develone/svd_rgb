In the current state this demo only represents doing work using mpich.
2 RPi3B+ and 2 RPi4 with 4GB were tested.
The images red.pgm, grn.pmg and blu.pgm were used 4 times on each node.
Changes are needed such that each threads uses a new image.
This results none the less are quite impressive. 
The next improvement would be to replace the 2 RPi3B+ with RPi4 4GB.
There would also be some improvement going to 64 Bit Raspbery Pi OS
